# Eyebot Revenue Waterfall Dashboard

A strategic revenue attribution dashboard for Eyebot's retail partners, showing complete customer journey from vision tests to glasses purchases with actionable recommendations.

## Tech Stack

- React 18 with TypeScript
- Vite
- Tailwind CSS
- Recharts
- Lucide React (icons)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is configured for Vercel deployment. Simply connect your repository to Vercel for automatic deployments.

## Features

- Complete revenue waterfall visualization
- Stage-by-stage drop-off analysis
- Actionable recommendations with ROI
- Performance benchmarking
- Prioritized optimization opportunities
